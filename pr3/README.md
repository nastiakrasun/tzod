# ⚡ Energy Consumption Calculation with Numba Optimization

## 📌 Опис

У проєкті реалізовано алгоритм обчислення енергетичних потреб на основі даних із датчиків, таких як потужність, температура, вібрація та коефіцієнт навантаження.

Основна увага приділяється оптимізації обчислень за допомогою бібліотеки Numba та порівнянню продуктивності з чистою реалізацією Python.

## 🎯 Мета

- Реалізувати алгоритм обчислення енергоспоживання
- Оптимізувати обчислення за допомогою Numba (@njit)
- Порівняти час виконання з базовою реалізацією Python
- Оцінити ефективність на різних розмірах даних

## 📂 Дані

Використовуються згенеровані дані датчиків:

- **power** — потужність
- **temperature** — температура
- **vibration** — рівень вібрації
- **load_factor** — коефіцієнт навантаження

## ⚙️ Методологія

Реалізовано два варіанти алгоритму:

- чистий Python (цикл for)
- оптимізований варіант з Numba

Час виконання вимірюється для різних обсягів даних

Побудовано графік залежності часу виконання від розміру даних

## 📈 Результати

Порівняння показує, що:

- Numba значно прискорює обчислення
- ефект оптимізації зростає зі збільшенням розміру даних
- для великих масивів даних спостерігається суттєвий приріст продуктивності

## 🧠 Висновок

Використання Numba є ефективним підходом для оптимізації чисельних обчислень у Python.
JIT-компіляція дозволяє значно зменшити час виконання алгоритмів, особливо при роботі з великими обсягами даних.
