# 📈 Виявлення аномалій у часових рядах (IQR)

## 📌 Опис

У проєкті реалізовано алгоритм виявлення аномальних значень у часовому ряді з використанням бібліотеки pandas та методу міжквартильного діапазону (IQR).

Було згенеровано часовий ряд із випадковими значеннями та додано штучні аномалії для демонстрації роботи алгоритму.

## 🎯 Мета

- Реалізувати виявлення аномалій у часовому ряді
- Використати міжквартильний діапазон (IQR)
- Візуалізувати результати та виділити аномальні точки

## 📂 Дані

Часовий ряд містить:

- **date** — дата спостереження
- **value** — значення показника

## ⚙️ Методологія

Обчислено квартилі Q1 та Q3

Визначено міжквартильний діапазон:

```
IQR = Q3 − Q1
```

Розраховано межі:

- нижня: Q1 − 1.5 × IQR
- верхня: Q3 + 1.5 × IQR

Значення поза межами визначаються як аномалії

## 📈 Результати

- Аномалії успішно виявлені та позначені
- Побудовано графік із виділенням аномальних точок
- Метод ефективно знаходить викиди у даних

## 🧠 Висновок

Метод IQR є простим і ефективним підходом для виявлення аномалій.
Він не потребує складних моделей і добре працює для базового аналізу часових рядів.
