# ⚡ Моделювання енергетичних потоків (SciPy)

## 📌 Опис

У проєкті реалізовано симуляцію енергетичних потоків у мережі за допомогою диференціальних рівнянь та бібліотеки SciPy.

Модель дозволяє аналізувати динаміку системи, оцінювати стабільність та досліджувати вплив параметрів (генерація, споживання, втрати) на поведінку мережі.

## 🎯 Мета

- Реалізувати модель енергетичних потоків через диференціальні рівняння
- Використати solve_ivp для чисельного розв’язання
- Оцінити стабільність системи при різних параметрах
- Порівняти з підходом на основі середніх значень

## 📂 Модель

Енергетичний стан системи описується рівнянням:

```
dx/dt = α − βx − γx²
```

де:

- **α** — генерація енергії
- **β** — коефіцієнт споживання
- **γ** — втрати

## ⚙️ Методологія

- Проведено симуляцію для різних наборів параметрів
- Побудовано часові графіки зміни енергії
- Стабільність оцінено через стандартне відхилення на кінцевій ділянці
- Виконано порівняння зі статистичним підходом (середні значення)

## 📈 Результати

- Модель показує динаміку системи та перехідні процеси
- Зміна параметрів суттєво впливає на стабільність мережі
- Диференціальний підхід дозволяє виявити коливання та нестабільні режими

## 🧠 Висновок

Моделювання за допомогою диференціальних рівнянь є більш інформативним, ніж аналіз середніх значень.
Воно дозволяє дослідити поведінку системи в часі та оцінити її стабільність при різних умовах.
