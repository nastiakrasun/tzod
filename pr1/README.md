# 📊 Аналіз пропусків та імпутація даних

## 📌 Опис

У проєкті продемонстровано обробку пропущених значень та оцінку їх впливу на ключову метрику — середній чек.

Було згенеровано штучний датасет із інформацією про товари (product), кількість (quantity), ціну (price) та знижку (discount). Частина значень навмисно відсутня для моделювання реальних умов.

## 🎯 Мета

- Проаналізувати пропущені значення (кількість і відсоток)
- Заповнити quantity та discount медіаною в межах product
- Порівняти середній чек до та після імпутації

## 📂 Дані

Датасет містить такі ознаки:

- **product** — категорія товару
- **quantity** — кількість
- **price** — ціна
- **discount** — знижка

## ⚙️ Методологія

Побудовано звіт по пропусках за допомогою Pandas

Середній чек обчислюється як:

```
quantity × price × (1 − discount)
```

Пропуски заповнюються медіаною в межах кожного product

Для повністю порожніх груп використано загальну медіану

## 📈 Результати

Порівняння середнього чеку до та після імпутації показує вплив обробки пропусків на агреговані показники.

Медіанна імпутація є стабільним методом, що зменшує вплив викидів і зберігає структуру даних.

## 🧠 Висновок

Обробка пропущених значень є важливим етапом аналізу даних.
Групова імпутація медіаною є надійним і доцільним підходом для таких задач.
