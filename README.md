# Проєкти з аналізу даних

Цей репозиторій містить серію проєктів з обробки та аналізу даних, реалізованих на Python з використанням різних бібліотек для наукових обчислень.

## Структура проєктів

- **pr1/**: Аналіз пропусків та імпутація даних — обробка пропущених значень та оцінка їх впливу на середній чек
- **pr3/**: Energy Consumption Calculation with Numba Optimization — оптимізація обчислень енергоспоживання за допомогою Numba
- **pr4/**: Моделювання енергетичних потоків (SciPy) — симуляція енергетичних потоків через диференціальні рівняння
- **pr5/**: Виявлення аномалій у часових рядах (IQR) — виявлення аномальних значень методом міжквартильного діапазону

Кожен проєкт має власний README.md файл у відповідній папці з детальним описом мети, даних, методології, результатів та висновків.

## Вимоги

- Python 3.x
- Необхідні бібліотеки (pandas, numpy, scipy, numba тощо) вказані та встановлюються у кожному проєкті окремо
